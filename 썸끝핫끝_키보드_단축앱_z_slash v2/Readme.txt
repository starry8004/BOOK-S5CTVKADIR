2. Chrome 브라우저에서 chrome://extensions로 이동합니다.
3. 브라우저에서 확장 프로그램 외부프로그램을 추가하는 개발자 모드를 켭니다 (크롬기준 우측 상단). 
4. "압축해제된 확장 프로그램을 로드합니다" 버튼을 클릭하고 압축 해제한 폴더를 선택합니다.
5. 썸끝 핫끝 페이지에서 좋음은 "Z" 나쁨은 "/"(slash,물음표자리)  
키를 누르면 됩니다.
